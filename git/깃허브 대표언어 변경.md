# 깃허브 대표언어(언어 비율) 변경



## 상황

스프링 프레임워크를 통해 백엔드 서버 공부를 하고 있다. 프론트엔드에는 아직 무지한 상태지만 맛깔난 디자인을 원해서 부트스트랩을 이용해 필요한 css,rcss 등을 적용시키고 커밋했더니 CSS가 가장 많은 언어 비율을 차지하게 됐다. 대표언어도 CSS로 변경되었다. 스프링(JAVA)을 이용해 만든 웹사이트이므로 해당 repository의 대표언어를 JAVA로 변경하고싶다.

## 해결

.gitattributes 파일을 생성하여 다음과 같이 작성한다.
```
*.css linguist-detectable=false
*.scss linguist-detectable=false
```

## 그 외
깃허브의 언어 비율은 파일 수가 아닌 파일 크기에 비례한다고 한다.
